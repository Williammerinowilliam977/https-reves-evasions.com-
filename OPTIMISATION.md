## 🚀 Installation et Lancement

### Prérequis
- Node.js installé (version 14+)
- npm (inclus avec Node.js)

### Étapes

1. **Installer les dépendances**
   ```bash
   npm install
   ```

2. **Lancer le serveur local**
   ```bash
   npm start
   ```
   ou
   ```bash
   npm run serve
   ```

3. **Accéder au site**
   - Le site s'ouvrira automatiquement à `http://localhost:8000`
   - Accessible aussi à `http://192.168.100.141:8000` (réseau local)
