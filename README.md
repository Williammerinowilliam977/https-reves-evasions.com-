# Thaïlande – Rêves & Évasions

Un site web magnifique et informatif sur les destinations touristiques en Thaïlande.

## Structure du Site

### Pages
- **index.html** - Page d'accueil avec présentation générale
- **bangkok.html** - Guide touristique de Bangkok  
- **chiangmai.html** - Guide touristique de Chiang Mai
- **phuket.html** - Guide touristique de Phuket
- **pattaya.html** - Guide touristique de Pattaya
- **ilesphiphi.html** - Guide touristique des Îles Phi Phi

### Fichiers de Style et Logique
- **style.css** - Feuille de style responsif
- **app.js** - Fonctionnalités interactives JavaScript
- **package.json** - Configuration npm

## Installation et Lancement

### Prérequis
- Node.js installé (pour utiliser http-server)

### Étapes

1. **Installer les dépendances**
   ```
   npm install
   ```

2. **Lancer le serveur local**
   ```
   npm run serve
   ```
   Ou en mode ouverture automatique:
   ```
   npm start
   ```

3. **Accéder au site**
   - Le site sera accessible à `http://localhost:8000`

## À Compléter

### 1. Images
Créer un dossier `images/` et ajouter:
- `logo-thailande.png` - Logo du site (220px de large)
- `fond-principal.jpg` - Image de fond pour le site
- Images destinations: `bangkok1.jpg`, `bangkok2.jpg`, etc.

### 2. Amélioration du Contenu
- Ajouter plus de détails sur chaque destination
- Créer une page de contact avec formulaire
- Ajouter une page tarification/offres spéciales

### 3. Fonctionnalités Futures
- Système de réservation complèt
- Formulaire de contact fonctionnel
- Section galerie photo interactive
- Blog ou articles de voyage

## Design et Couleurs

- **Couleurs principales**: 
  - Rouge foncé: #8B0000
  - Or: gold
  - Vert: #008000
  - Noir semi-transparent pour les fonds

- **Typographie**: Police Georgia (serif) pour un style élégant

## Responsive Design

Le site s'adapte automatiquement aux écrans mobiles, tablettes et ordinateurs de bureau.

## Notes

- Assurez-vous que toutes les images sont optimisées pour le web
- Testez le site sur différents navigateurs
- Vérifiez la compatibilité mobile avant le déploiement
